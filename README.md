# command_piping
Execute a command and get result back as a string.

## Example

```c++
cout << RDI::execute_command("cowsay hi") + '\n';
```

output:

```
 ____
< hi >
 ----
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||
```
